Guide d’installation de l’extension Proton Pass

🔧 Installation pour Vivaldi

Vivaldi utilise le même moteur que Chrome, vous pouvez donc installer l’extension depuis le Chrome Web Store.

  1. Ouvrez Vivaldi et rendez‑vous sur le Chrome Web Store – Proton Pass.
  2. Cliquez sur le bouton « Ajouter à Chrome ». Vivaldi vous demandera confirmation.
  3. Acceptez les autorisations requises, puis cliquez sur « Ajouter l’extension ».
  4. L’icône de Proton Pass apparaît dans la barre d’outils. Cliquez‑la pour créer ou importer votre coffre‑fort de mots de passe.
  5. Suivez les instructions à l’écran pour synchroniser vos mots de passe avec votre compte Proton Pass.

🦊 Installation pour Firefox

Firefox possède son propre catalogue d’extensions. Voici comment installer Proton Pass :

  1. Ouvrez Firefox et allez sur la page officielle de l’extension : Proton Pass – Firefox Add‑ons.
  2. Cliquez sur le bouton « Ajouter à Firefox ».
  3. Une boîte de dialogue s’affichera ; vérifiez les permissions puis choisissez « Ajouter ».
  4. L’icône de Proton Pass apparaîtra dans la barre d’outils de Firefox.
  5. Cliquer sur l’icône pour vous connecter à votre compte Proton Pass ou créer un nouveau coffre‑fort.
